
## 介绍
nndeploy是一款支持多平台、高性能、简单易用、模块化的机器学习部署框架。一套算法的实现可以在多端(云、边、端)完成模型的高性能部署，将AI推理部署简洁的抽象为Task，将会提供较为丰富且通用的模型前后处理，当前一个应用具有多个模型是常态，可将多个Task组成有向无环图的方式完成多模型的高性能部署，目前会着重服务[HuggingFace]的模型落地。
